Open an Existing File

This section will show you how to open an existing file from your disk.

Drag & Drop

Cuties also support drag and drop gestures. You can drag your source file into the main window of Cuties, Cuties will also load the file. That's the most simple way to open a file.

From menu

At any time, you can open a file from the menu. Click the main button and choose File menu, then Open. Browsing your file, select it, then OK. And Cuties will load the file and use the corresponding syntax highlighter by the suffix.

From toolbar

When you have already opened a file, the toolbar will be shown. And the second tool button of the toolbar is open a file. This button just work the same as the 'Open' command in File menu.

From history record

Cuties will record your open history in the sidebar. Double click one of the history record. If the file is still existing, Cuties will automatic load the file. For more information about sidebar and history, see Sidebar and History Recording
