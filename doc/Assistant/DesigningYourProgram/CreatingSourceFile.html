Create Source File

This section will show you how to create a new file in Cuties.

Create a new file from Main Menu

You can create a new blank file from Main Menu or Main Menu Bar at any time. From Main Menu, click New Source File in File menu. Cuties will automatically create a new source file using your default programming language syntax highlight.

Create a new file from Tool Bar

You can create a new file form the toolbar as well, but the toolbar will only visible when there's a file have already opened. Click ? button and Cuites will create a new source file. The setting of the new document is just the same as the new file create via Main Menu.
