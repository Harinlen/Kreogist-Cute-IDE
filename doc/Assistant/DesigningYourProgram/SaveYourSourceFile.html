Save Your Source File

This section will show you how to save your source file in Cuties.

Save your source code file

If you have save the current document before, you will find the file is named as 'Untitled X*'. X is a number that count how many file you have create from Cuties executes. To save your file, press the home button, select File menu, and then Save. If you are using a Mac version, press the File menu on the menu bar, then select Save.

Then choose a directory to save your code with a new name. You can see the caption of the tab has changed into your new file name when you finish the saving. The asterisk(*) will shown when you do any modifies to your document.

Now, your file has been already saved. This time when you press the Save command like what you do above, your source code file will automatic save all the modifies. The asterisk will disappear to show that the file has been saved.

Save all the opened files

As you know, Cuties supports opening several files at the same time. You can edit them and you may want to save them all. You can use the 'Save All' command in the File menu. And Cuties will automatic save all the modified documents for you.

Exit Cuties with document modified

You may forget to save your source files and you press the 'x' button or press the 'Exit' command. Cuties will check whether there's any unsaved file, and prompt you to save them.
