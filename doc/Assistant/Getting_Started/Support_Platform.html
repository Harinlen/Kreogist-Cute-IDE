Supported Platforms

You can install and run Kreogist Cuties on several operating systems to create applications on multiple desktop.

Developments Platforms

Kreogist Cuties is available in binary packages for the following operating systems:

  • Windows XP/Vista/7/8 (32-bit and 64-bit)

  • (K)Ubuntu Linux 10.04 (32-bit and 64-bit) or later

  • Mac OS 10.5 or later
