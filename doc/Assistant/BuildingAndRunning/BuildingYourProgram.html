Building your program

This section will show you how to compile your source code in Cuties and execute the binary.

Compile your source code

In Cuites, you don't need to compile your source code, link, or build execution manually. We provide three function to help you compile and build execution automatically:

  • Compile and Run
  • Compile
  • Run

Compile and Run

Compile and Run command can compile your file via the compiler which you set or provided by default, and launch kciConsoleRunner to execute the binary.
