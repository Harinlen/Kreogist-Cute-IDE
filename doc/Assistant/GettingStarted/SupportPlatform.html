Supported Platforms

You can install and run Kreogist Cuties on several operating systems to create applications on multiple desktop.

Developments Platforms

Kreogist Cuties is available in binary packages for the following operating systems:

  • Microsoft® Windows®

  • Ubuntu Linux

  • Mac OS X

    Reference Configurations

    a reference configuration is a combination of several compilers, a debugger and some other tools as a default development suit for a platform. Here are the configurations that are tested by the Kreogist Dev Team:

    PlatformCompiler
    Ubuntu Linux 13.04, X11 (64-bit)As provided by Ubuntu
    Microsoft Windows 7 (32-bit and 64-bit)MinGW-builds gcc 4.8.1 (32-bit)
    Microsoft Windows 8 (32-bit and 64-bit)MinGW-builds gcc 4.8.1 (32-bit)
    Apple Mac OS X 10.8 "Mountain Lion", Cocoa (64-bit)GCC as provided by Apple

    You can also use your own configuration to write your code. Please see Compiler Configuration for further information.

    Availability of Binary Installation Packages

    Cuties provides binary installation packages for all of the platforms above from https://github.com/Kreogist/Cuties/releases. The site provides download links for all supported development platforms.
