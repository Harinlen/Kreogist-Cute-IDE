Getting Started Guides

This section describes how to use Cuties to write code, and this will show you how to create a simple C code. You can also test that your Cuites installation is successful by running this simple example.

Create a New Empty File

Use the main menu in Windows/Linux or main menu bar in Mac to create a new file. For more information about creating a new file, see Creating Source File

Write your code

Write your code in the Code Editor. Here we provide a simple example:

#include <stdio.h>
int main(int argc, char *argv[])
{
	printf("Hello World!\n");
	return 0;
}

This is a very simple C program to display 'Hello World!' on the screen. You can type this or copy this to Cuties. And save it as a C Source File. For more information about saving the source file, see Save your source file.

Compile and Run

You can compile the file and run the execution manually. But you can just click 'Compile and Run' button on the tool bar. For more information about compile and run, see Building your program.

Now there is a console window launched with four lines: the first line is 'Hello World!' text, the second line is a blank line, the third line shows you the executing time, and the cursor is at the last line, prompt you press any key(Windows)/enter(Others) to exit console runner.
