Overview

What is Cuties?

Kreogist Cuties is an integrated development environment (IDE) that provides you with tools to develop program for ACMers/OIers. It's designed for developing programs on serveral desktop system. Supports C/C++ and Pascal. Cuties is engineered to deliver the most innovating and simplified programming experience as a single file compiling, debugging and executing development environment.

Why use Cuties?

Here's what you can get in Cuites:

A all new design User Interface.

Come and try our brand new user interface. You will never forget it, and get enjoyed with programming.

The same awesome experience on all platform.

Built on rock-solid Qt foundation, Cuties is built to deliver the most intuitive and integrated programming experience on all the mainstream platform.

Super easy compile and run.

Compile and run your program with a simple click. We also provide you a binary executor to reckon you the program execution time, and the executor will pause the terminal window so that you don't need to pause your program mannually.

Easy to learn? No need to learn.

Everything in Cuties is designed to be intuitive and super easy to use. You have no need to learn how to write your code in Cuties. It just works.
