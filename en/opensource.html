Project Cuties


Cuties is a open source project. We hope that everyone could have chance to build Cuties. And we believes that using Open Source methodology could make Cuites a more robust, steady IDE. Any problems found with this software can be immediately identified and fixed by Kreogist Dev Team and the Open Source community.

Cuties is available under The GNU General Public License v3.0 and GNU Lesser General Public License v3.0. Learn more about licenses here.
You can get split source packages from below. The source code is also available as a single zip file for Windows users or as tar.gz file for Linux/Mac users. Or visit the repository at Project Cuties home page.

Cuties source code tar.gz package(master): tar.gz format source code package of branch master.
Cuties source code zip package(master): zip format source code package of branch master.
Cuties source code tar.gz package(dev): tar.gz format source code package of branch dev.
Cuties source code zip package(dev): zip format source code package of branch dev.
